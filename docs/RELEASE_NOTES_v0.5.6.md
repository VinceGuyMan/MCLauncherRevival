# MCLauncherRevival v0.5.6-alpha Release Notes

v0.5.6-alpha is a usability and historical-authenticity polish release. It keeps the nostalgic
launcher style intact while making first-run guidance clearer and reducing broad fallback notes for
old versions.

## Highlights

- Added exact static note entries for more launcher-visible maintenance/reupload builds, so fewer
  selected versions need broad family fallbacks.
- Kept sparse historical builds honest with cautious wording instead of invented patch details.
- Tightened historical era presentation:
  - Alpha is more compact/login-board flavored.
  - Indev/Infdev is more prototype/dev-note flavored.
  - Classic and Pre-Classic use smaller, simpler chrome.
- Expanded the first-run guide with clearer online/offline/XP/release-ZIP guidance.
- Added a Profile Editor shortcut to reopen the first-run guide.
- Added small control tooltips for common actions without changing the classic visual layout.
- Preserved browser/OAuth login safety: the launcher should never ask for a raw Microsoft password.

## Compatibility notes

- Windows 7-11 remains the primary full-auth target.
- Windows XP remains offline/classic focused.
- macOS and Linux remain experimental for old Minecraft/LWJGL game launch.
- Use the attached GitHub Releases ZIP for normal play, not GitHub's source-code ZIP.
