# Trust and Safety

## What this launcher is

MCLauncherRevival is an unofficial, alpha-quality nostalgia project for a classic 2011-style
Minecraft launcher experience.

It is intended for preservation, nostalgia, and learning.

## What this launcher is not

- It is not the official Minecraft Launcher.
- It is not affiliated with Mojang, Microsoft, Xbox, or Minecraft.
- It is not a way to bypass Minecraft ownership or entitlement.
- It is not a password collector.
- It is not a place for ads, telemetry, crypto, tracking, or unrelated monetization.

## Account safety expectations

The launcher should never ask for a Microsoft password inside the app.

Where online login is available, sign-in should happen through a browser/OAuth flow. Saved tokens or
settings should remain local to the user's machine.

Users should review the source code before trusting any alpha launcher with account-related flows.

## Downloads

Users should download runnable artifacts only from this repository's GitHub Releases page. Source
checkouts and GitHub's automatic source archives are for review and local builds.

Asset origin and licensing status are recorded in [Asset Provenance](../ASSETS.md). Files whose
origin is not yet verified must not be presented as project-owned or covered by the MIT license.

## Project maintenance expectations

Contributions should stay readable, focused, and reviewable.

Do not add:

- Obfuscated code.
- Malware or account-stealing behavior.
- Telemetry or tracking.
- Ads or spammy marketing links.
- URL shorteners.
- Unrelated SEO keywords.
- Suspicious workflows that upload files to third-party locations.
