## Third-party services

The launcher can contact Microsoft/Xbox/Minecraft authentication and profile services when the user
chooses Microsoft Login.

The launcher can contact Mojang/Minecraft version metadata and library download endpoints when
versions are launched or downloaded.

## Local token cache

The launcher may store OAuth tokens in the user's local `.minecraft` folder so the user does not
have to log in every time.

It should not ask for or store raw Microsoft passwords.
