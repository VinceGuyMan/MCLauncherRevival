# Notices

MCLauncherRevive is an unofficial launcher revival and modernization project.

## Not affiliated

This project is not affiliated with Mojang, Microsoft, Xbox, Minecraft, Scrolls/Caller's Bane, or Cobalt.

## Third-party services

The launcher can contact Microsoft/Xbox/Minecraft authentication and profile services when the user chooses Microsoft Login.

The launcher can contact Mojang/Minecraft version metadata and library download endpoints when versions are launched or downloaded.

## Local token cache

The launcher stores OAuth tokens in the user's local `.minecraft` folder so the user does not have to log in every time.

It does not ask for or store raw Microsoft passwords.

## Historical assets and artifacts

Some files in this project may exist to preserve the look and compatibility of the old launcher. Ownership of Minecraft-related names, artwork, launcher artifacts, services, and downloads remains with their respective owners.
